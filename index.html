Cómo Usar Esta Aplicación

📋 Paso 1: Información

Completa los datos del cliente (nombre, email y teléfono).

📄 Paso 2: Contrato

Selecciona el tipo de contrato y pega el contenido. Verás una vista previa automática.

🖐️ Paso 3: Huella Digital

Haz clic en "Capturar Huella Digital". Tu dispositivo te pedirá usar tu sensor biométrico (huella o cara). Acepta para capturar tu biometría.

✓ Paso 4: Firmar

Una vez capturada la huella, haz clic en "Firmar y Guardar". Se guardará la huella + contrato en tu dispositivo.

📭 Galería

Todas tus firmas se guardan en la galería. Puedes ver, descargar o eliminar cada una. También puedes descargar todas como un ZIP.

🔒 Seguridad

Las firmas digitales se almacenan de forma segura en tu repositorio privado de GitHub como archivos JSON. El token de acceso solo se guarda localmente en tu navegador y nunca abandona tu dispositivo.

💾 Almacenamiento

GitHub permite repositorios de hasta 1 GB. Cada firma ocupa pocos KB. Configura tu repositorio y token en la pestaña "Configuración".
